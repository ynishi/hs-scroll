# hs-scroll
* require xdotool
* cabal
* ghc-8.10

## install
* If you don't setup ghc with version, remove cabal.project file and try. GHC version 8 is supported, 9 is not work.
```
git clone ...
cd ...
cabal install
```
## run
* via cabal or run executable
### via cabal
```
cabal run
```
### run executable
```
# cli
hs-scroll
# gtk
hs-scroll-gui
```
